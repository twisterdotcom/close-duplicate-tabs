Close Duplicate Tabs

Settings & Configuration

Duplicate Detection Settings

Configure how the extension identifies duplicate tabs.

When enabled, URLs with different query strings are treated as duplicates.
Example: example.com/?a=1 and example.com/?b=2 are considered duplicates.

When enabled, URLs are compared without their hash/anchor, but the hash is still used to determine which tab to keep (highest anchor number).
Example: example.com/#comment-123 and example.com/#comment-456 are duplicates, but the one with #comment-456 is kept.

When enabled, remaining tabs will be automatically reloaded after closing duplicates.

Choose which AI provider to use for tab organization. The selected provider must have a valid API key configured.

How to Use

Click the extension icon in your Chrome toolbar to instantly close duplicate tabs.

The extension will:

  • Detect duplicate tabs based on your settings
  • Keep the tab with the highest anchor number (e.g., latest GitHub comment)
  • Close all other duplicates

The extension icon badge shows how many duplicate tabs are currently open.

AI Tab Organization: Use the "Organize Tabs" button in the popup to automatically group your tabs using AI. Configure your API keys above to enable this feature.

Settings are automatically saved.
